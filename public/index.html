Documentação da API

A API de tradução permite traduzir texto entre diferentes idiomas. Abaixo estão os endpoints disponíveis:

Endpoint de Tradução

POST /api/v1/translate
Parâmetros do Corpo (JSON)
  • text (obrigatório): O texto a ser traduzido
  • targetLang (opcional): O código do idioma de destino (padrão: "en")
  • sourceLang (opcional): O código do idioma de origem
Exemplo de Requisição
{
  "text": "Hello world",
  "targetLang": "pt",
  "sourceLang": "en"
}
Exemplo de Resposta
{
  "originalText": "Hello world",
  "translatedText": "Olá mundo",
  "detectedLanguage": "en",
  "targetLanguage": "pt",
  "translationTime": 125.45,
  "timestamp": "2023-05-01T12:34:56.789Z"
}
